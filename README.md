FPGA-based ultrasonic phased-array imaging front-end using VHDL, implementing parallel beamforming and beam-steering modules for real-time signal-processing applications.

```markdown
# FPGA Ultrasonic Phased Array Imaging Front-End

## Overview

This repository contains the VHDL design files for an FPGA-based ultrasonic phased-array imaging front-end developed for real-time signal-processing applications.

The project focuses on the implementation of parallel hardware architecture for ultrasonic beamforming and beam steering, with the goal of supporting real-time object localisation and echolocation. The design was developed as part of an Electronic Engineering honours-level project exploring FPGA acceleration for signal-processing systems.

The repository includes individual VHDL component modules that form part of the overall signal-processing pipeline, including timing/control logic, beam-steering support, and parallel processing structures.

## Key Focus Areas

- FPGA-based digital system design
- VHDL component-level development
- Parallel hardware architecture
- Ultrasonic phased-array signal processing
- Beamforming and beam-steering concepts
- Real-time embedded hardware implementation

## Tools / Technologies

- VHDL
- FPGA development workflow
- Intel Quartus / ModelSim
- Digital signal-processing architecture

## Project Status

This repository is intended as a portfolio and reference project. The design files represent component-level FPGA modules developed for an ultrasonic phased-array imaging front-end.
```
